**학습 목표 및 요구사항**

- jsx 문법에 맞는 어트리뷰트 사용
- 어트리뷰트 방식의 이벤트 핸들러 익숙해지기

- object 형태의 값을 상태로 이용한 여러개의 인풋 동시 관리
- useRef 훅을 이용한 돔 요소 관리
